"Windows - Device/Credential Guard issue during VMware Workstation launch"

Published: Sun 16 July 2017

In content.

If you are prompted with following error:

Determine if you are running EFI or BIOS: launching cmd as administrator type msinfo32 in the prompt

I am using legacy BIOS so the fix will be running following command at prompt: bcdedit /set hypervisorlaunchtype off

For EFI visit following link:https://kb.vmware.com/selfservice/microsites/search.do?language=en_US&cmd=displayKC&externalId=2146361
