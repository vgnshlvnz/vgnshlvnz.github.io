"[TUTORIAL] Install Xfce4 on Linux-Debian"

Published: Sun 17 July 2016

In content.

Update

Wrote this article when I was using Linux Mint

This tutorial is written on how to remove existing desktop environment and install xfce4.

Brief Introduction

Xfce4 is a light weight desktop environment with low memory footprint. Using Xfce4 is somewhat like using a mac, just a little bit older mac.

To remove desktop environment, e.g cinnamon:

1) apt-get autoremove --purge cinnamon

Now install Xfce4

2)apt-get install xfce4 xfce4-goodies

Reboot your workstation shutdown -r now

On reboot, select xfce4 desktop environment somewhere near your username/login place.
